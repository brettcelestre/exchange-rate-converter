# Exchange Rate Converter

> A single page app for converting exchange rates based on services provided by [Open Exchange Rates](https://docs.openexchangerates.org/).

## Features

* AngularJS
* Node/Express
* ES6
* ESLint
* SASS
* Webpack
* Gulp

## Application Organization

```
client/
  assets/
    css/
    sass/
  app/
    components/
      home/
    shared/
      converter/
    app.module.js
    app.routes.js
  bundle.js
  index.html
server/
  config/
  rates/
  server.js
tests/
```

## Getting Started

``` bash
# install dependencies
npm install --save
bower install --save

# serve at localhost:8080
npm start

# run all tests
npm test
```
